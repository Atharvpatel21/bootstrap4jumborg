Go to Package Control’s installation page and copy the Sublime Text 3 python code. In Sublime Text 3, go to View, Show Console. Paste in the python code and hit Enter. Then, press command+shift+p to bring up the Command Palette. Type in “Package Control” and select Package Control: Install Package.
